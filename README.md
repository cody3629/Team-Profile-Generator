## Description

This project was developed for a homework assignment. The goal of this project was to receive input from the user creating their software team, and then outputting the information onto a webpage. This was developed using Node.js, JavaScript, HTML, CSS, and Jest.

## Installation

1.) Install node.js 2.) Clone code into your repository 3.) In the terminal, enter 'npm install inquirer' 4.) In terminal, run the command 'node app.js' and answer the prompts to develop your software dream team 5.) Go to output folder and open index.html in your browser and you will see your software dream team!
